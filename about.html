Useful - Bot

Useful is a discord bot that makes your server looking professional looking!

What Useful has to offer to your discord server

  • Welcome and leave messages
  • Admin commands
  • Fun & Game Commands(Working Progress)
  • Moderation Commands
  • Support tickets

Administrator Commands

  • !clear (number of messages to clear here!
  • !ban (reason; Up to 20 words) - bans players
  • !kick (reason; Up to 20 words) - kicks players and you can give a reason
  • !announce - puts announcements in a fancy box like this one
  • !setwchannel - Set welcome message channel
  • !setlchanel - Set Leave message channel
  • !setwdescription - set Welcome message description
  • !setautorole (RoleID) - Set what role a member gets when they join. Note: You must use the role's id
  • !enabletickets - enables members to use tickets
  • !poll - create a voting poll
  • !say - make the bot say something
  • !deletebadwords (amount) - Deletes messages with badwords

Support Chat Commands

  • !new - creates a support ticket
  • !close - Closes a support ticket.

Other Commands

  • !members - Tells how many members are on the server
  • !invitebot - Invite this bot to a discord server
  • !invite - Get server invite
  • !meme - Produces a random meme
  • !love (mentioned) - Give someone love
  • !slap (mentioned) - Slap someone
  • !poke (mentioned) - Poke someone
  • !askout (mentioned) - Ask someone out
  • !breakup (mentioned) - Break up with someone

Minecraft Related Commands

  • !status - Checks any Minecraft Server's status
  • !skin - Get a player's minecraft skins
